Class: WorldObject

WorldObject

new WorldObject(posX, posY, scaleX, scaleY, rot, sprite, game, group, leftCollideable)

WorldObject is an object intended to be the platforms and obstacles in the game which the player can can interact with. I.E Obstacles kill the player and Platforms allow the player to walk on them there is nothing that sets the two apart other than how they interact with the player in the game loop
Parameters:
Name Type Description
posX number position of WorldObject on the x plane
posY number position of WorldObject on the y plane, positive is down
scaleX number width scale of the WorldObject sprite
scaleY number height scale of the WorldObject sprite
rot number rotation of the WorldObject sprite
sprite string string that's a key of an image loaded in via Phasers load.image function
game Phaser.Game phaser game object to allow access to game specific functions
group Phaser.Group link to a phaser group object to allow all the WorldObject objects to be added to a group
leftCollideable boolean a boolean that dictates if the player can collide with its left side, true for yes, false for no that allows for easy collision checking etc.!
