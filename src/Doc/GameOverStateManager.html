Class: GameOverStateManager

GameOverStateManager

new GameOverStateManager()

Instantiates the GameOverStateManager which instantiates the goodGameOverScreen, gameOverScore and badGameOverScreen states and allows the user to switch between them and return each of the states

stateSwitch(game, gameOverState, playGameState)

Function that switches between the GameOverStateManager's GameOver States, it accepts the state you wish to switch to as a parameter and switches to it and exits the previous state as long as it wasn't null
Parameters:
Name Type Description
game Phaser.Game a link to the instantiated Phaser Game Object so we can pass it into the create function of the GameOver States
gameOverState GameOverScreenState | GameOverScoreState a link to the GameOver State we wish to switch too
playGameState Game a link to the current game session our own Game object not phasers, so I can pass it into our GameOver States
