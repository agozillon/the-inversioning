Class: Init

GameState. Init

new Init()

Creates the Init Object, which is essentially used as a state class within Phaser It's job is to setup the games window, other states and load in the assets

Methods

<private> create()

Function for the Init state used to setup the games window and position the games objects within the world (as we can see them in all state transitions)

<private> preload()

Function for pre-loading all of the assets relevant to the rest of the game such as images and music
