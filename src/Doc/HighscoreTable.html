Class: HighscoreTable

HighscoreTable

new HighscoreTable(game)

Creates the Highscore Table Object, which is a table that holds the top 10 scores in the game, renders them on screen as Phaser Text objects, has a function to pull them from an online databases and push them to an online database
Parameters:
Name Type Description
game Phaser.Game a link to an instantiated Phaser Game Object

Methods

add(name, score, arrayPosition)

Function that adds a score to the table which consists of two separate arrays holding a name and a score and a phaser text object
Parameters:
Name Type Description
name string name of the current player playing
score number number that denotes the players score that's getting added to the table
arrayPosition number number that denotes the Highscore table number to replace with the new addition in the list

createText()

Function that creates all of the Highscore text as Phaser text objects capable of being rendered to screen it also positions them appropriately on the screen for them to be positioned as a highscore table, all phaser objects get deleted on state switch

getHighscoreTable()

Function that pulls all of the highscore data from the google app engine server. There can be up to 10 unique values stored at a time (1 for each position)

isItAHighscore(score) → {boolean}

Function that returns true if the score is a highscore or false if its not a highscore on the table
Parameters:
Name Type Description
score number number that denotes the players score that we're checking against the table
Returns:
Type
boolean

postHighscore(pos)

Function that uses Ajax to send a highscore value to the Google app engine server
Parameters:
Name Type Description
pos number number that defines the position of the highscore in the table it'll be used as a unique key by the server replacing any previous highscore with that key
