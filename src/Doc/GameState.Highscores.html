Class: Highscores

GameState. Highscores

new Highscores()

Creates the Highscores Object, which is essentially used as a state class within Phaser It's job is to display the Highscore Screen that displays all the highscores.

Function that Initializes all things required for the Highscores screen and table, Phaser takes care of it's own renders and Phaser specific its own objects updates.
